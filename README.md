# ECHacksBack

## Main

- The production branch has the live code running on the AWS ec2 instances.

The backend system for our ECHacks project

## Installation

- Install python3
- Install required third-party packages
  - ```pip3 install -r requirements.txt```

## Usage

```python3 server.py```
